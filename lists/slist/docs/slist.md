# Singly-linked Lists #
---

## Description ##
A singly-linked list is a type of linked list where each node keeps track of the node in front of it.

## Operations ##
- Insert
  - After
- Remove
  - After
- Traverse
  - Forward
- Length

## Edge Cases ##
- Deletion of head
- Deletion of tail
- Deletion of nodes not in the list
- Deletion of null nodes

## Hierarchy ##

### Parent ###
Linked lists

### Children ###
None
